## 학습 목표

- 시맨틱 HTML(= 의미 있는 태그를 사용하는 HTML 작성 방식) 구조 설계
- CSS 레이아웃 (Flexbox, Grid) 숙련
- 바닐라 JS(= 순수 JavaScript, 라이브러리 없이 작성)로 DOM 조작
- 실제 서비스 수준의 UI 재현 능력 향상

---

## 기술 스택

- **HTML5** - 시맨틱 마크업
- **CSS3** - Flexbox, Grid, Animation
- **Vanilla JavaScript** - DOM 조작, 이벤트 처리
